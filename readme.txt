Simple share buttons for Facebook, Twitter, and Google+.

== Description ==

Well-written theme-agnostic widget that provides single-style social “like” buttons.
	
* Setup in seconds. No coding skill required.
* Only three services: Facebook, Twitter, and Google+. That’s it.
* Vertical or horizontal orientation.
* Super skinny, it’s just a styled links, no heavy scripts and iframes.
* Accessible for tabbing, right-, and wheel-clicking.
* Seamless responsive. Retina-ready. Neat.
* Beautiful flat skins with fast CSS3 transitions on hover, press, and focus.

Licensed under MIT. You are free to use Sharerama in commercial projects.

== Installation ==

1. Install Sharerama either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin and go to “Widgets” page.
3. Drag Sharerama Widget to a sidebar or click on it. Play with the options. Save.
